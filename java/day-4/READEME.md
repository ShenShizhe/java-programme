#### Demo0
1. 计算1……100的和,for循环
```
  		int sum=0;
  		for(int i=1;i<=100;i++) {			
  		sum+=i;
  		}			
```

		
2. 计算1……100的和，while循环
```
    int sum=0;
	  int i=1;
		while(i<=100) {
			sum+=i;
			i++;
		}
```
		
		
3. 1~100不能被3整除的数的和
```
      int sum=0;
      for(int i=1;i<=100;i++) {
      if(i%3!=0) 
          sum+=i;
      }	
```
    
		
4. 计算1……10的阶层和(1！+2！……9！+10！)
```
		long t=1,sum=0;
		for(int i=1;i<=10;i++) {
			t*=i;	
			sum+=t;
		}
```
